Что нужно сделать ДО встречи

4 шага

Займёт 10–15 минут. Сделайте это заранее — на встрече сразу перейдём к делу.

1

💻 Открыть терминал

Терминал — это текстовой интерфейс вашего компьютера. Здесь будет работать Claude Code.

🍎 Mac
  • Cmd+Space → напечатать "Terminal" → Enter
  • Или: Finder → Программы → Утилиты → Terminal
🪟 Windows
  • Win+R → напечатать "cmd" → Enter
  • Или: Поиск → "PowerShell" → Enter
# Примерно так выглядит терминал: user@MacBook ~ % _
2

📦 Установить Claude Code

Скопируйте команду и вставьте в терминал (Ctrl+V или Cmd+V), затем нажмите Enter:

~ % npm install -g @anthropic-ai/claude-code # После установки проверьте версию: ~ % claude --version claude 1.x.x
💡
Если npm не установлен Сначала скачайте Node.js с nodejs.org — нажмите кнопку LTS (рекомендуемая версия). После установки Node.js команда npm станет доступна.
3

🔑 Подключить аккаунт Anthropic

Запустите Claude — он откроет браузер для авторизации:

~ % claude # Браузер откроется автоматически. # Войдите в свой аккаунт Anthropic. # Вернитесь в терминал — увидите: ✓ Authenticated successfully. Welcome!
🔐
Войдите с аккаунтом, на котором у вас подписка Claude Max ($100/мес). Это даёт огромный лимит токенов — достаточно для серьёзной автоматизации. Claude Code — это отдельный инструмент от claude.ai в браузере, но использует тот же аккаунт.
4

⚡ Режим без подтверждений опционально

Для скорости можно запустить Claude без запросов подтверждения на каждое действие:

~ % claude --dangerously-skip-permissions
⚠️
Только для опытных пользователей Этот режим позволяет Claude выполнять команды без вашего подтверждения каждый раз. Удобно для скорости, но вы принимаете на себя ответственность за все действия. На первой встрече мы будем использовать обычный режим.
